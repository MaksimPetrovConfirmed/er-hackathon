1. Навык Алисы (Вся дока тут: https://tech.yandex.ru/dialogs/alice/doc/about-docpage/) (4 + баллов)
	- Настройка и развертывание веб сервиса, приложение типа пинг/понг (1 бал)
	- Настроить работу с сетью: Подключить api Дом ру для работы с нашим каталогом контента (1 бал)
	- Реализовать голосовй поиск контента  (Пример команд: "Найди фильмы с Анджелиной Джоли"/"Хочу смотреть форсаж" и т.д.)(1 бал)
			* распознаем Яндекс Диалогами
			* делаем запрос на наше апи
			* выводим в интерфейсе Алисы
	- Реализоать возможность воспроизведения контента (1 бал) 
	- Реализовать адаптивную речевую модель (типо ИИ :))): т.е. можно использовать синоными, переставлять слова местами, использовать разные словосочетания (пример "Найди фильм ..."/"Хочу посмотреть ...") (Чем больше ИИ тем больше баллов))

			* https://github.com/yandexmobile/yandex-speechkit-samples-android
	- Настроить работу с сетью: Подключить api Дом ру для работы с нашим каталогом контента (1 бал)
	- Реализовать голосовй поиск контента (Пример команд: "Найди фильмы с Анджелиной Джоли"/"Хочу смотреть форсаж" и т.д.) (1 бал)
			* распознаем SpeechKit-ом
			* делаем запрос на наше апи
			* выводим как-нибудь в интерфейсе
	- Реализовать интеграцию с любым плеером (MediaPlayer, ExoPlayer) и возможность запуск контента с помощью голосовых команд (Запусти "Что-то"/Воспроизвести) (1 бал)
	- Реализовать адаптивную речевую модель (типо ИИ :))): т.е. можно использовать синоными, переставлять слова местами, использовать разные словосочетания (пример "Найди фильм ..."/"Хочу посмотреть ...")  (Чем больше ИИ тем больше баллов))
